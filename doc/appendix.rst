Appendix
========

.. envvar:: HOMEPAGE

   Package's home page, see :manpage:`ebuild(5)` and variables_ in `Gentoo
   Development Guide`_.

.. envvar:: P

   Package name and version, see :manpage:`ebuild(5)` and variables_ in `Gentoo
   Development Guide`_.

.. envvar:: PN

   Package name, see :manpage:`ebuild(5)` and variables_ in `Gentoo Development
   Guide`_.

.. envvar:: RESTRICT

   Portage features to disable for given ebuild, see :manpage:`ebuild(5)` and
   variables_ in `Gentoo Development Guide`_.

.. envvar:: PORTAGE_GPG_KEY

   gnupg_ key to sign commits with, see :manpage:`make.conf(5)`.

.. envvar:: PORTDIR_OVERLAY

   Location of extra ebuild repositories, see :manpage:`make.conf(5)`.

.. _variables: http://devmanual.gentoo.org/ebuild-writing/variables/index.html
.. _Gentoo Development Guide: http://devmanual.gentoo.org/
.. _gnupg: http://www.gnupg.org/
